📋 Mode d'emploi complet

1
Sur votre PC, lancez le serveur local : double-cliquez sur LANCER-SERVEUR.bat (Windows) ou exécutez python server.py dans le terminal.
2
Ouvrez http://localhost:8080 dans votre navigateur. Déposez votre vidéo dans la zone ci-dessus.
3
Une URL locale s'affiche (ex: http://192.168.1.XX:8080/tv). Ouvrez-la sur chaque télé via leur navigateur intégré.
4
La vidéo se lit en boucle automatique, en plein écran, sans interface. Gardez votre PC allumé pendant la diffusion.
