📋 Comment utiliser

1
Déposez votre vidéo dans la zone ci-dessus (MP4 recommandé).
2
Sur chaque télé, ouvrez un navigateur et allez sur cette même URL, puis cliquez sur « Lecteur TV ».
3
Note importante : chaque télé doit charger le fichier depuis le même ordinateur/réseau. Ce site est un lanceur — la vidéo reste locale. Hébergez le site via GitHub Pages pour l'URL, mais la vidéo se charge localement sur chaque poste.
4
La vidéo se lit en boucle automatique, sans son par défaut (pour respect des règles navigateur), plein écran.
